# ai-science-alphafold

The data used in this study were downloaded on July 7, 2025, and are publicly available. The readers could download the data into the `data/raw` folder and run the code in the `code/build` to reproduce the results. Or use the data after processing in the `data/processed` folder, which we provide in this repository. The code for analyzing the dataset is implemented in Stata 18 and is available in the `code/regression`.
